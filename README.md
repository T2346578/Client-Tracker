# Prime Shine CRM (no-install)

This is a simple **client + revenue + invoice tracker** that runs as a static website.

## How to run

- Open `index.html` in your browser (Chrome/Edge recommended).
- Your data is saved locally in your browser (Local Storage) on this device.

## What it does

- Track **Clients** (name + contact info)
- Track **Jobs/Services** (date + description + amount)
- See **Monthly** and **Yearly** totals
- Create **Invoices** and **Print / Save as PDF**
- **Export/Import** your data as a JSON backup

## Notes

- If you clear browser storage, your data will be deleted. Use Export regularly.
- If you want this hosted online later, we can deploy it as a static site (no server required).
